# Contributing to Koodap

Thank you for your interest in contributing! Here's how to get involved.

## Ways to Contribute

- 🐛 Report bugs via [GitHub Issues](https://github.com/YOUR_USERNAME/koodap/issues)
- 💡 Suggest features via [GitHub Discussions](https://github.com/YOUR_USERNAME/koodap/discussions)
- 🔧 Submit pull requests for bug fixes or new features
- 📖 Improve documentation
- 🌍 Add translations

## Development Setup

Follow the installation steps in [README.md](README.md).

## Pull Request Guidelines

- One feature or fix per PR
- Include a clear description of what changed and why
- Test on both Android and iOS if possible
- Keep changes focused — avoid unrelated cleanup in the same PR

## Code Style

- TypeScript for all React Native code
- Python for server-side code
- No comments unless the WHY is non-obvious
- Descriptive variable and function names

## Reporting Security Issues

Please do **not** open public issues for security vulnerabilities. Email the maintainer directly.

## Code of Conduct

Be kind, inclusive, and constructive. We follow the [Contributor Covenant](https://www.contributor-covenant.org/).
